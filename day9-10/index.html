HTML & CSS Basics Summary

HTML Summary

HTML(HyperText Markup Language)is used to define our page content,structure and meaning you don't use it for styling purpuses.Use CSS for that instead

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag,content and then closing tag
  • you can also have void(empty) elements like images
  • you can also configure elements with attributes
  • there is long lists of avaleble elements but you'll gain experience over time,no worries

Learn more about all avaleble HTML elements on The MDN HTML elements refrences

CSS Summary

CSS(Cascading Style Sheets)is used for styling your page content.

  • styles are asigned via property-value pairs
  • you can assign via style via the 'style' attribute
  • to evoid code duplication,you tipically uses global styles(e.g. via the 'style'element)insted
  • Alternatively, you can work with external stylesheet files wich you "link" to
  • when working with CSS , concepts like "inheritance","specificty"and the "box model" should be understood

Learn more about all avaleble CSS propestis and values on The MDN CSS property refrences
